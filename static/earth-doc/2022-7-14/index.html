前端八股题集

一、前端基础

  • html,css,js有哪些命名规范？

    html和css采用-加小写字母的命名方式，js使用驼峰命名方式，类用大驼峰，函数用小驼峰。

  • css优先级

    !important(慎用)>内联>style
    id(#div)>class(.div)>标签(div)

  • js类新标准#私有变量和约定的_私有变量有什么区别？

    #不会被继承，_会被继承

  • restful api 设计思路是什么？

    请求 = 动词 + 宾语，动词使用五种 HTTP 方法，对应 CRUD 操作；宾语 URL 应该全部使用名词复数，可以有例外，比如搜索可以使用更加直观的 search ；如果记录数量很多，API应该提供参数，过滤返回结果，?limit=10 指定返回记录的数量 ?offset=10 指定返回记录的开始位置
